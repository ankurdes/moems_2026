📋 How to Run This Practice Round (Teacher-Driven)

  1. One screen, both teams in the same room. The two teams of 5 work simultaneously on paper at separate tables. You (the teacher) hold the laptop with this page.
  2. Click ▶ Start to reveal all 10 problems. Read each problem aloud (or hand out a printed copy). No timer — work at whatever pace fits the room.
  3. For each question, type both teams' answers in the red and blue boxes, then click "✓ Grade Q#" directly below that question. The system instantly shows ✓/✗ next to each team's answer, reveals the correct answer, and — if either team got it wrong — auto-reveals the full step-by-step explanation right there for class review.
  4. Running scores at the top update after every question you grade.
  5. Optional: the "✓ Grade All Remaining" button at the top grades any remaining ungraded questions in one go (useful for cleanup at the end).

Coaching tip: Encourage runners to call out clean, simplified answers (e.g., "twelve" or "12", not "12 square units"). The grader is forgiving (handles fractions, decimals, dashes, etc.) but cleaner is better.

Click "Show solution" on each problem to review with the class.
